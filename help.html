About the project

The Optic Lobe CNS Edition distills connectomic insights into concise narratives. Each section pairs descriptive context with high-resolution diagrams, providing an offline-friendly reference for researchers, educators, and students.

Key features

  • Layered anatomy

    Step through retinotopic layers and observe how signals propagate across lamina, medulla, and lobula structures.

  • Cell-type gallery

    Browse representative neurons with morphology callouts, synapse counts, and concise behavioral annotations.

  • Data highlights

    Static infographics summarize network motifs, dominant pathways, and experimentally validated checkpoints.
